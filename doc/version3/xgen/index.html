Overview

XGen is a source code generator that uses the Velocity template engine. Loaders and templates have been build to generate:

  • JDBC persistence classes
  • POJO plain old java objects
  • Xml2Html conversions

News

November 2004

XGen Documentation is updated and released.

April 2004

XGen is refactored and re-released under the workzen code tree. XGen now supports multiple template instances, allowing complex code builds. The persistence code has been seperated from individual classes, and support  for persistence of references and collections has been added.

July 2002

Initial release of XGen. XGen builds souce code that allows bean level persistence of objects.
