Quick Start

  • Download XGen
  • Create an input file
  • Create an Ant build file
  • Run Ant

Scenario

Typically, I will explode the XGen distribution into the top level of my project tree, and create Ant build files for all source code needs. As the project changes, I can change the input schema, and regenerate my JDBC code as needed.

Hint: The JDBC manager classes are often customized by hand, so comment out your manager tasks to prevent overwriting.
