JavaModel Loader

The JavaModel Loader loads an xml schema into a Package model. The model provides access to all classes and attributes specified in the schema.

Java Code Generation: POJO

Generate plain old java beans.

Loader workzen.xgen.loader.JavaModelLoader
Model workzen.xgen.model.java.Package
Template templates/pojo/control.vm
Function generate Plain Old Java beans

Configuration

<xgen
  loader="workzen.xgen.loader.JavaModelLoader"
  inputFile="${input.file}"
  resourceTemplate="templates/pojo/control.vm"
  templatePath="${basedir}"
  outputDirectory="${basedir}/work/"
  outputFile=""
  >
</xgen>  

JDBC Code Generation: JDBC Persistence

The persistence classes combine POJO / table level classes with POJO manager classes. See technical for more information.

Loader workzen.xgen.loader.JavaModelLoader
Model workzen.xgen.model.java.Package
Template templates/persistence/manager/control_manager.vm
Template templates/persistence/manager/control_table.vm
Function generate persistence classes for Plain Old Java beans

Configuration

Xml Code Generation

Generate xml schema from a Package model.

Loader workzen.xgen.loader.JavaModelLoader
Model workzen.xgen.model.java.Package
Template templates/schema/xgen.vm
Function generate xml schema from package model

Configuration

<!-- generate transitive schema -->
<xgen
   loader=""
   inputFile="${basedir}/etc/input/blog.xml"
   resourceTemplate="templates/schema/xgen.vm"
   templatePath="${basedir}"
   outputDirectory="${basedir}/work/schema/blog"
   outputFile="blog.xml"
   >
</xgen>
