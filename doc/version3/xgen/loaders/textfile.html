TextFile Loader

The TextFileLoader loads text into the TextFile model. The TextFile model performs findAndReplace() on specified tokens. This model can be used to perform Xml2Html type transformations.

Xml2Html Transformation

Loader workzen.xgen.loader.TextFileLoader
Model workzen.xgen.model.file.TextFile
Template templates/text/htmlEscape.vm
Function replace tokens in the original file

Configuration

<xgen
   loader="workzen.xgen.loader.TextFileLoader"
   inputFile="${basedir}/etc/input/blog.xml"
   resourceTemplate="templates/text/htmlEscape.vm"
   templatePath="${basedir}"
   outputDirectory="${basedir}/work/xml2html/"
   outputFile="blog_schema_1.html"
   >
</xgen>
