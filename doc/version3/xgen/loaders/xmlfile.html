XmlFile Loader

The XmlFileLoader loads xml into the XmlDoc model. The XmlDoc model performs getHtmlOutput() on the document. This model can be used to perform Xml2Html type transformations.

Xml2Html Transformation

Loader workzen.xgen.loader.XmlDocLoader
Model workzen.xgen.model.xml.XmlDoc
Template templates/text/htmlOutput.vm
Function use JDom to transform the file

Configuration

<xgen
   loader="workzen.xgen.loader.XmlDocLoader"
   inputFile="${basedir}/etc/input/blog.xml"
   resourceTemplate="templates/xml/htmlOutput.vm"
   templatePath="${basedir}"
   outputDirectory="${basedir}/work/xml2html/"
   outputFile="blog_schema_2.html"
   >
</xgen>
