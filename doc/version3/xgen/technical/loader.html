Model Loaders

Loaders load models for use with velocity templates. Templates use the top level model object to perform the transformation. See ILoader.java.

TextFileLoader

 loader="workzen.xgen.loader.XGenLoader"
  inputFile="${basedir}/etc/input/student.xml"
  resourceTemplate="templates/pojo/control.vm"
  templatePath="${basedir}"
  outputDirectory="${basedir}/work/"
  outputFile=""
 

Property Value
loader IFileLoader package
inputFile absolute path to xml input file
resourceTemplate relative path to velocity template
templatePath non null value searches "file" before "classpath"
outputDirectory directory for all output files
outputFile optional output file if not using a "control" template
