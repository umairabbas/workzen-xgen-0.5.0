Configuration

Ant Configuration

Below are the parameters for the XGen task. See the individual loaders for more detailed information.

Property Value
loader ILoader package name
inputFile absolute path to input file
resourceTemplate relative path to velocity template
templatePath (optional) non null value searches "file" before "classpath"
outputDirectory directory for all output files
outputFile (optional) output file if not using a "control" template

Example Ant Schema

<!-- convert xml to html using htmlEscape.vm and the TextFileLoader -->
<xgen
  loader="workzen.xgen.loader.TextFileLoader"
  inputFile="${basedir}/etc/input/blog.xml"
  resourceTemplate="templates/text/htmlEscape.vm"
  templatePath="${basedir}"
  outputDirectory="${basedir}/work/xml2html/"
  outputFile="blog_schema_1.html"
  >
</xgen>
